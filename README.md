## EuropPool contract

EuroPool is a simple staking application. Deposit, wait, earn

## Usage

### Setup

```shell
$ make all
```

### Install

```shell
$ make install
```

### Run local chain

```shell
$ make anvil
```

### Run Tests

```shell
$ make test
```

### Deploy

```shell
$ make local-deploy
```

or

```shell
$ make alfajores-deploy
```
